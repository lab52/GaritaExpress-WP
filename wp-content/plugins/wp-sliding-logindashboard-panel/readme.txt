== Description ==

Add a sliding login/dashboard panel to Wordpress Theme. This awesome panel is based on http://web-kreation.com/index.php/tutorials/nice-clean-sliding-login-panel-built-with-jquery/

Instead of modifying directly theme files, just install this plugin so you can preserve you panel when switching between themes and during themes updates.

You can also specify your own HTML content to put in the panel

== Installation ==

1. Upload `wp-sliding-logindashboard-panel` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
